The Ad Manager Plugin is a simple to use which allows you to create custom ads for posts, pages and sidebars.



==  Description ==


 
Create custom advertisement which can be placed on your pages, posts, sidebars. Thanks to simple shortcodes and widgets. You can design the image size for your advertisement and also provide target links which will redirect users to different websites.

Ad Manager create responsive ads automatically so you don't need to worry about mobile responsive.


Features:
1. Very simple to Use
2. Create as many ads as you need. There is no limit on it.
3. Monitor clicks on your advertisement so you can create report of it.
4. Edit anytime you want.
5. Create ads for posts, pages and sidebars.
6. Supports JPEG, PNG & GIF files.

== Installation ==
1. Download the Ad Manager plugin from WordPress.org as Zip file and install it using Upload Plugin option.
2. Alternatively you can directly place the plugin folder under "/wp-content/plugins/" directory and activate it from the plugins list. Find for Ad Manager in the plugins list.
3. Click on "Ad Manager" menu in left navigation
4. It is ready to create new advertisements.
